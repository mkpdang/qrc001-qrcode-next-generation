## Environment

- **Package manager:** Use `uv` for all Python dependency and virtual environment management.
- **Venv location:** `.venv` in project root (created via `uv venv`)
- **Install packages:** `uv pip install <package>`
- **Add dependencies:** `uv add <package>`
- **Run scripts:** `uv run python <script>`
- **Never use** `pip install` or `python -m venv` directly.
